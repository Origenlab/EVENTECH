## La Luz Es Emoción: Transforma Cualquier Espacio

La iluminación es el elemento más poderoso y menos comprendido de la producción de eventos. Una luz bien diseñada puede hacer que un salón simple luzca espectacular, mientras que una mala iluminación arruina hasta el venue más lujoso.

En EVENTECH somos especialistas en [iluminación profesional para eventos](/servicios/iluminacion/). Esta guía te convertirá en un conocedor capaz de comunicar exactamente lo que quieres.

---

## Las 4 Capas de Iluminación Profesional

Todo diseño de iluminación profesional se construye en **4 capas**. Domina este concepto y entenderás cualquier rider técnico.

```
┌─────────────────────────────────────────────────────────────┐
│                    CAPA 4: EFECTOS ESPECIALES               │
│            Humo, láser, confeti, sparklers, strobo          │
├─────────────────────────────────────────────────────────────┤
│                    CAPA 3: ESCENARIO/SHOW                   │
│          Moving heads, follow spots, barras LED             │
├─────────────────────────────────────────────────────────────┤
│                    CAPA 2: DECORATIVA                       │
│         Fairy lights, candiles, letras, esferas             │
├─────────────────────────────────────────────────────────────┤
│                    CAPA 1: ARQUITECTÓNICA (BASE)            │
│            Wash, uplighting, bañadores, ambiente            │
└─────────────────────────────────────────────────────────────┘
```

| Capa | Función | % del Presupuesto | Imprescindible |
|------|---------|-------------------|----------------|
| [Arquitectónica](/servicios/iluminacion/arquitectonica/) | Define el espacio | 30-40% | Sí, siempre |
| [Decorativa](/servicios/iluminacion/decorativa/) | Crea atmósfera | 25-35% | Sí, en sociales |
| [Escenario](/servicios/iluminacion/escenario/) | Dinamismo visual | 20-30% | Según evento |
| [Efectos](/servicios/iluminacion/efectos-especiales/) | Momentos wow | 10-20% | Opcional premium |

## Capa 1: Iluminación Arquitectónica

La [iluminación arquitectónica](/servicios/iluminacion/arquitectonica/) es la base de todo. Define colores, resalta estructuras y establece el mood general.

### Tipos de Luminarias Arquitectónicas

| Luminaria | Función | Ángulo | Potencia | Uso Principal |
|-----------|---------|--------|----------|---------------|
| [Wash LED](/servicios/iluminacion/arquitectonica/wash-lights/) | Bañar superficies | 25-60° | 100-300W | Paredes, techos |
| [Uplighting](/servicios/iluminacion/arquitectonica/uplighting/) | Iluminar hacia arriba | 15-30° | 50-150W | Columnas, muros |
| [Bañadores](/servicios/iluminacion/arquitectonica/banadores/) | Líneas de luz | 10-45° | 50-200W | Texturas, fachadas |
| [Fachadas](/servicios/iluminacion/arquitectonica/fachadas/) | Exterior edificios | Variable | 200-500W | Arquitectura exterior |

### Técnicas de Uplighting

El [uplighting](/servicios/iluminacion/arquitectonica/uplighting/) es la técnica más impactante con menor inversión:

```
    ┌───────────────────┐
    │                   │
    │    ▲    ▲    ▲   │  ← Luz proyectada hacia arriba
    │    │    │    │   │
    │   ░░░  ░░░  ░░░  │  ← Luminarias en piso
    └───────────────────┘
         PARED/COLUMNA
```

**Posicionamiento óptimo:**

| Altura del Muro | Distancia de Luminaria | Ángulo |
|-----------------|------------------------|--------|
| 3m | 30-50cm del muro | 15° |
| 4m | 40-60cm del muro | 20° |
| 5m | 50-80cm del muro | 25° |
| 6m+ | 60-100cm del muro | 30° |

### Cantidad de Luminarias por Espacio

| Espacio | Perímetro | Luminarias Recomendadas | Espaciado |
|---------|-----------|------------------------|-----------|
| Salón pequeño | 30m | 10-15 uplights | 2-3m |
| Salón mediano | 50m | 16-25 uplights | 2-3m |
| Salón grande | 80m | 26-40 uplights | 2-3m |
| Jardín | Variable | 1 por árbol/punto focal | — |
| Carpa | Perímetro | En cada poste + paredes | 2-3m |

## Capa 3: Iluminación de Escenario

La [iluminación de escenario](/servicios/iluminacion/escenario/) agrega dinamismo y profesionalismo. Es lo que diferencia un evento "bonito" de uno "espectacular".

### Tipos de Luminarias de Escenario

| Luminaria | Función | Movimiento | DMX Canales |
|-----------|---------|------------|-------------|
| [Par LED](/servicios/iluminacion/escenario/par-led/) | Color de base | Fijo | 4-7 |
| [Moving Head Beam](/servicios/iluminacion/escenario/moving-heads/) | Haces de luz | Pan/Tilt | 16-24 |
| [Moving Head Spot](/servicios/iluminacion/escenario/moving-heads/) | Proyección gobos | Pan/Tilt | 18-32 |
| [Moving Head Wash](/servicios/iluminacion/escenario/moving-heads/) | Baño de color | Pan/Tilt | 16-20 |
| [Follow Spot](/servicios/iluminacion/escenario/follow-spots/) | Seguimiento | Manual | 1-4 |
| [Barras LED](/servicios/iluminacion/escenario/barras-led/) | Efectos lineales | Fijo | 8-48 |

### Par LED: La Base del Show

Los [Par LED](/servicios/iluminacion/escenario/par-led/) son las luminarias más versátiles:

| Variante | LEDs | Colores | Uso |
|----------|------|---------|-----|
| RGBW | 18-24 | Rojo, Verde, Azul, Blanco | Estándar |
| RGBWA | 18-24 | + Ámbar | Tonos cálidos |
| RGBWA+UV | 18-24 | + Ultravioleta | Efectos neón |
| COB | 1 LED grande | Variable | Alta potencia |

**Posicionamiento de Par LED:**

```
         ESCENARIO/PISTA
    ┌───────────────────────┐
    │                       │
    │                       │
    │         ⬤            │  ← Cenital (desde arriba)
    │                       │
    └───────────────────────┘
    ⬤                     ⬤   ← Front wash (desde frente)

    ⬤   ⬤   ⬤   ⬤   ⬤   ⬤   ← Back light (detrás de escenario)

    ⬤ = Par LED
```

## Control DMX: El Cerebro de la Iluminación

### ¿Qué es DMX?

DMX512 es el protocolo estándar para controlar iluminación profesional. Cada luminaria ocupa "canales" que controlan diferentes funciones.

| Canal | Función Típica |
|-------|----------------|
| 1 | Intensidad (dimmer) |
| 2 | Rojo |
| 3 | Verde |
| 4 | Azul |
| 5 | Blanco |
| 6 | Pan (movimiento horizontal) |
| 7 | Tilt (movimiento vertical) |
| 8+ | Gobos, zoom, efectos |

### Consolas de Iluminación

| Nivel | Consola | Canales | Uso |
|-------|---------|---------|-----|
| Básico | DMX controller simple | 192 | Eventos pequeños |
| Intermedio | Chamsys QuickQ | 512 | Bodas, corporativos |
| Profesional | GrandMA2 onPC | 4096+ | Conciertos, TV |
| Premium | GrandMA3 | 250,000+ | Tours, festivales |

### Modos de Operación

| Modo | Descripción | Ideal Para |
|------|-------------|------------|
| Manual | Operador controla en vivo | Shows dinámicos |
| Timecode | Sincronizado con música | Videos musicales |
| Automático | Secuencias preprogramadas | Sin operador |
| Sound active | Responde al audio | Fiestas |

## Errores Comunes y Cómo Evitarlos

| Error | Consecuencia | Solución |
|-------|--------------|----------|
| Pocos uplights | Espacios oscuros | Mínimo 1 cada 2-3m |
| Color incorrecto | Fotos con mal tono | Prueba con fotógrafo |
| Sin dimmer | Luz muy intensa | Siempre controla intensidad |
| Cables visibles | Aspecto descuidado | Usa canaletas/cinta |
| Sin operador | Iluminación estática | Contrata técnico |
| Sin backup | Falla arruina evento | 10% equipo extra |
| Humo excesivo | Activa alarmas | Coordina con venue |
