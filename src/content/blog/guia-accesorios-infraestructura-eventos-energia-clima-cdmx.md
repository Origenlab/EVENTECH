# La Infraestructura Invisible que Hace Posibles los Eventos Memorables

Detrás de cada evento exitoso existe una **infraestructura invisible** que los invitados nunca notan—hasta que falla. La electricidad que alimenta el sonido, el clima que mantiene el confort, los textiles que definen la estética, y la señalización que guía el flujo. Esta guía te convierte en experto en los elementos que profesionales experimentados consideran **más críticos que el entretenimiento mismo**.

### Fórmula de Cálculo de Potencia Total

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    FÓRMULA DE POTENCIA TOTAL                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   Potencia Total = Σ(Potencia × Factor Pico) × Factor Seguridad        │
│                                                                         │
│   Donde:                                                                │
│   • Factor Seguridad = 1.25 (margen 25%)                               │
│   • Factor Pico = Arranque de motores y compresores                    │
│                                                                         │
│   ──────────────────────────────────────────────────────                │
│                                                                         │
│   EJEMPLO: Boda 200 personas exterior                                   │
│                                                                         │
│   Sonido:      8,000W × 1.5 = 12,000W                                  │
│   Iluminación: 7,500W × 1.1 = 8,250W                                   │
│   Catering:    9,700W × 1.5 = 14,550W                                  │
│   Clima:       6,000W × 1.8 = 10,800W                                  │
│   Pista LED:   2,500W × 1.2 = 3,000W                                   │
│                              ─────────                                  │
│   Subtotal:                  48,600W                                    │
│   × Factor Seguridad 1.25 =  60,750W                                   │
│                                                                         │
│   PLANTA RECOMENDADA: 75 kW (siguiente tamaño disponible)              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### Cálculo de Climatización por Carpa

```
┌─────────────────────────────────────────────────────────────────────────┐
│                CÁLCULO DE CLIMATIZACIÓN PARA CARPAS                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   FÓRMULA ENFRIAMIENTO:                                                 │
│   BTU = (Área m² × Factor Zona) + (Personas × 600) + Equipos           │
│                                                                         │
│   Factor Zona CDMX:                                                     │
│   • Sombra total: 250 BTU/m²                                           │
│   • Semi-sombra: 350 BTU/m²                                            │
│   • Sol parcial: 500 BTU/m²                                            │
│                                                                         │
│   ──────────────────────────────────────────────────────                │
│                                                                         │
│   EJEMPLO: Carpa 200m² para 150 personas                               │
│                                                                         │
│   Área:     200m² × 300 BTU/m² = 60,000 BTU                            │
│   Personas: 150 × 600 BTU      = 90,000 BTU                            │
│   Equipos:  Sonido + Luces     = 20,000 BTU                            │
│                                 ────────────                            │
│   Total:                        170,000 BTU                             │
│                                                                         │
│   EQUIPO: 2 × AC 10 ton (240,000 BTU) ✓                                │
│                                                                         │
│   ──────────────────────────────────────────────────────                │
│                                                                         │
│   FÓRMULA CALEFACCIÓN:                                                  │
│   BTU = Área m² × Factor Aislamiento × ΔT                              │
│                                                                         │
│   Factor Aislamiento:                                                   │
│   • Carpa cerrada con paredes: 15 BTU/m²/°C                            │
│   • Carpa semi-abierta: 25 BTU/m²/°C                                   │
│   • Carpa abierta: 40 BTU/m²/°C                                        │
│                                                                         │
│   EJEMPLO: Carpa 200m², elevar de 10°C a 22°C (ΔT=12)                  │
│                                                                         │
│   BTU = 200 × 20 × 12 = 48,000 BTU/hr                                  │
│                                                                         │
│   EQUIPO: 1 × Calefactor 80K BTU ✓                                     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## ⚠️ Errores Comunes y Cómo Evitarlos

### Error #1: Subestimar la Potencia Eléctrica

```
❌ INCORRECTO:
"Sumamos el consumo de cada equipo y pedimos esa planta"

✅ CORRECTO:
Multiplicar por factores de pico + 25% margen seguridad
```

**Consecuencia:** Planta se sobrecarga, se apaga todo a media fiesta.

### Error #2: Ignorar el Clima

```
❌ INCORRECTO:
"Es diciembre, no necesitamos climatización"

✅ CORRECTO:
Bodas nocturnas en diciembre REQUIEREN calefacción (temp < 15°C)
```

**Consecuencia:** Invitados incómodos, se van temprano.

### Error #3: Textiles de Último Momento

```
❌ INCORRECTO:
"Los manteles los conseguimos el mismo día"

✅ CORRECTO:
Reservar textiles mínimo 2 semanas antes, confirmar color/cantidad
```

**Consecuencia:** Colores que no combinan, cantidades insuficientes.

### Error #4: Señalización Improvisada

```
❌ INCORRECTO:
"Ponemos cartulinas escritas a mano"

✅ CORRECTO:
Señalización profesional diseñada con la imagen del evento
```

**Consecuencia:** Imagen poco profesional, invitados perdidos.
