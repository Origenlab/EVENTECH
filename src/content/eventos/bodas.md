#### Para la Ceremonia
- Sillas para invitados (Tiffany, Crossback o Avant Garde)
- Arco o estructura para altar
- Alfombra o camino
- Pedestales decorativos
- Iluminación ambiental

#### Para la Recepción
- Mesas para invitados (redondas o rectangulares)
- Mesa de honor para novios

#### Para la Fiesta
- Pista de baile (LED, madera o vintage)
- Sistema de sonido profesional
- Iluminación ambiental y de fiesta
- Salas lounge para descanso
